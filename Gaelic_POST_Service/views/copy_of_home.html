Gaelic Part - of - Speach  Tagger API

Gaelic PoS tagger API was developed in conjunction between the Edinburgh Napier University and the University of Edinburgh.

 

PoS uses the tagger produced by Celtic and Scottish Studies at the University of Edinburgh and makes its methods available through a REST API.

 

To try the service right now, you can use the Demo page.

 

 

If you want more information about the project, visit the About page.

If you're a developer interested in this API, visit the Documentation page to learn more about how to use it.
