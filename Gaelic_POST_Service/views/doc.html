The API exposes 2 methods over the internet: tokenising and tagging.

Each one can work on a string passed as a parameter or on a file.

 

Click on a method to know more about it.

 

• /tokenise/string

• /tokenise/file

• /tag/default/string

• /tag/default/file

• /tag/simplified/string

• /tag/simplified/file

 

 

How to Use Gaelic Tagger API

Tokenise a string - /tokenise/string

Input (POST parameters):

 

• text: the text to tokenise

• ip: if this parameter is present, we will log your IP

• application: the name of the application using the API

 

Returns a JSON array containing the tokens

Example

Result: ['Dè', 'an', 't-', 'ainm', 'a', 'tha', 'ort']

Tokenise a file - /tokenise/file

Input (POST parameters):

 

• file: the file to tokenise (standard HTTP file transfert)

• ip: if this parameter is present, we will log your IP

• application: the name of the application using the API

 

Returns a JSON array containing the tokens

Example

Result: ['An', 't-', 'urram', 'thar', 'gach', 'beinn', 'Aig', 'Beinn', 'Dòbhrain', ';', 'De', 'na', 'chunnaic', 'mi', 'fon', 'ghrèin', ',', "'S", 'i', 'bu', 'bhòidhche', 'leam', '...']

Tag a string using the default tagger - /tag/default/string

Input (POST parameters):

 

• text: the text to tag

• ip: if this parameter is present, we will log your IP

• application: the name of the application using the API

 

Returns a JSON array containing token / tag couples.

Example

Result: [['Dè', 'Uq'], ['an', 'Tdsm'], ['t-', 'Uo'], ['ainm', 'Ncsmn'], ['a', 'Q-r'], ['tha', 'V-p'], ['ort', 'Pr2s']]

Tag a file using the default tagger - /tag/default/file

Input (POST parameters):

 

• file: the file to tag (standard HTTP file transfert)

• ip: if this parameter is present, we will log your IP

• application: the name of the application using the API

 

Returns a JSON array containing token / tag couples.

Example

Result: [['An', 'Spa-s'], ['t-', 'Uo'], ['urram', 'Ncsmd'], ['thar', 'Sp'], ['gach', 'Dq'], ['beinn', 'Ncsfd'], ['Aig', 'Sp'], ['Beinn', 'Ncsmd'], ['Dòbhrain', 'Nt'], [';', 'Fi'], ['De', 'Ncsmn'], ['na', 'Q-r'], ['chunnaic', 'V-s'], ['mi', 'Pp1s'], ['fon', 'Spa-s'], ['ghrèin', 'Ncsfd'], [',', 'Fi'], ["'S", 'Wp-i'], ['i', 'Pp3sf'], ['bu', 'Ws'], ['bhòidhche', 'Ncsfn'], ['leam', 'Pr1s'], ['...', 'Fb']]

Tag a string using the simplified tagger - /tag/simplified/string

Input (POST parameters):

 

• text: the text to tag

• ip: if this parameter is present, we will log your IP

• application: the name of the application using the API

 

Returns a JSON array containing token / tag couples.

Example

Result: [['Dè', 'U'], ['an', 'Td'], ['t-', 'U'], ['ainm', 'Nc'], ['a', 'Q'], ['tha', 'V-p'], ['ort', 'Pr']]

• application: the name of the application using the API

 

Returns a JSON array containing token / tag couples.
